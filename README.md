# GAP — Geometric Active Probe (GAAS Active Probe)

本リポジトリは、わらしべ長者の幾何学的拡張である。100万から10京に至るまで、大数の操作はすべて、空間の歪み（負圧）を利用した最小作用の帰結として記述される。

**趣旨の詳細:** [docs/PURPOSE.md](docs/PURPOSE.md) — 三角関数によるエネルギー流測定と特異点・相転移のブレイクスルー発見の一般則

---

## 概要

GAP（Geometric Active Probe）は、対象を「特定の周波数を持つ球体（質点）」として定義し、その周波数照射と最小神経確立によって、コストゼロの**負圧ポイント（特異点）**を発見・通過するための階層的質量操作システムである。

操作対象は、小規模な個体（$10^6$ 単位）から、都市規模（$10^{10}$ 単位）、地球・国家規模（$10^{16}$ 単位）を経て、全事象の総体（$10^{17}$〜京単位）までを同一の数理モデルで扱う。

## 質量階層（Layer）

| Layer | スケール範囲 | 名称 | 選定条件 |
|-------|-------------|------|----------|
| 1 | $10^6$ - $10^8$ | Micro Sphere | 高密度・高流動、固有の弾性係数が極めて高い |
| 2 | $10^9$ - $10^{10}$ | Medium Sphere | 組織的な慣性（慣性モーメント）を持ち、一定の負圧を発生させ得る |
| 3 | $10^{11}$ - $10^{13}$ | Macro Sphere | 広域の重力場を形成し、他球体の軌道に影響を与える大規模質量 |
| 4 | $10^{14}$ - $10^{17}$ | Global Sphere | 空間の曲率そのものを規定し、全事象の校正の基準となる観測点 |

## モジュール構成

- **FrequencyEmitter** — 周波数照射：対象のスケールに応じたパルスを生成し、同調（Resonance）を確認
- **ImpedanceFinalizer** — 最小神経確立：対象の輪郭を物理的に確定し、不変の幾何形状へ固定
- **VacuumPathFinder** — 負圧ポイント発見：抵抗ゼロ座標を抽出
- **TransitionExecutor** — ゼロコスト通過：負圧経路を確定し、エネルギー消費なしで座標を置換

## 重要ルール

1. **大数操作の線形性** — 小規模で実証された負圧通過ロジックは、スケールアップしても同一アルゴリズムを使用する
2. **待機の定式化** — 特異点が計算上出現していない場合、一切の演算と行動を停止（Idle State）し、観測精度のみを高める
3. **負圧の優先** — 「押し出す」のではなく、「吸い込まれる」状態が確立された時のみ `EXECUTE` する

## 使用例

```python
from gap import GeometricActiveProbe
from gap.constants import Layer

probe = GeometricActiveProbe()
result = probe.operate(target_mass=1e8, layer=Layer.MICRO)
```
